\section*{Introduction}

When the Commander X16 encounters a problem while running your program, it
displays an error message. Understanding these messages will help you fix
problems in your programs quickly.

\subsection*{?SYNTAX ERROR}
The X16 doesn't understand the command or statement. Check for typos and missing punctuation.

\subsection*{?ILLEGAL QUANTITY ERROR}
A number is out of the acceptable range for the command being used.

\subsection*{?OUT OF MEMORY ERROR}
The program is too large or has used all available memory.

\subsection*{?UNDEF'D STATEMENT ERROR}
A GOTO or GOSUB refers to a line number that doesn't exist.

\subsection*{?TYPE MISMATCH ERROR}
You tried to use a string where a number was expected, or vice versa.

\subsection*{?FILE NOT FOUND ERROR}
The file you tried to load doesn't exist on the SD card.

\subsection*{?DIVISION BY ZERO ERROR}
You attempted to divide a number by zero.
